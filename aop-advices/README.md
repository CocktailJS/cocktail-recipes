#AOP-ADVICES example

This is a very simple example about how to use the **advisable** trait.

> MyClass.js

This is a simple class with the advisable trait and on method named `initialize` intended to showcase the advices execution.

> index.js

Here we create an instance of MyClass and set some advices (after and before) over the initialize method (the adviced method).

## Install dependencies

```
npm install

```

## Run

```
node index.js
```
